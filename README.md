# Image Watermark Adder

A desktop application built with Python, Tkinter, and Pillow that allows users to add custom text watermarks to images and save the result locally.

## Features

* Upload an image from your computer
* Add custom watermark text
* Choose watermark size
* Choose watermark color
* Choose watermark position:

  * Top Left
  * Top Right
  * Bottom Left
  * Bottom Right
  * Center
* Preview the watermarked image before saving
* Save the final image to your preferred location

## Technologies Used

* Python
* Tkinter
* Pillow (PIL)

## How to Run

1. Clone the repository:

```bash
git clone <repository-url>
```

2. Install dependencies:

```bash
pip install pillow
```

3. Run the application:

```bash
python main.py
```
